CSS Media Queries

CSS media queries are often used to help apply specific styles to code based on the user's device and/or screen type. To work, they check for specific conditions, such as layout or screen size, and apply CSS rules only if the specific conditions are met. This helps to make a responsive design for a website, allowing it to change and adapt to different devices from phones, tablets, desktops, etc. Without media queries, the website will unlikely display properly on different user environments.

Within media queries, there are a range of media features that help target specific conditions. There are a multitude of different options to chose from when specifying the parameters for the display of a website. There are common features such as width and orientation, and other useful ones like aspect-ratio and overflow-block.

  • Width
    • This feature targets the width of the viewport or screen. It is often used with conditions such as min-width or max-width to control a layout and how it would adjust across different screen sizes. We often see this change when rotating a phone from portrait to landscape or from mobile to desktop.
  • Aspect-Ratio
    • This feature compares the width and height of the screen. It specifically allows the application of styles dependent on whether the screen is more square, wide, or tall. It can be especially useful for adjusting layouts or images to fit different screen proportions better.
  • Orientation
    • This features helps detect if a device or screen in in portrait or landscape mode. It helps change the layout depending on how the user is holding the device, such as better organizing the content to fit either a wider or taller screen.
  • Overflow-Block
    • This feature refers to how the content of a page behaves when it overflows from the grid, most often seen with vertical scrolling. It helps developers to adjust the style no matter if the content is clipped or scrollable. This is especially helpful for improving readability of a layout on different screens.
